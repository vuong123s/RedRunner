## Kiến thức đã học được
  Hiểu cấu trúc project Unity (Assets, Scenes, Scripts, Prefabs).
  Biết cách sử dụng Prefab để tái sử dụng GameObject.
  Hiểu cách thay đổi thông số gameplay trong Inspector.
  Biết cách thêm object vào Scene và kiểm tra hoạt động trong Play Mode.
  Làm quen với Unity Editor và script C# trong game development.
